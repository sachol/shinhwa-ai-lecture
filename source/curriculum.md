## 모듈 1: 오프닝 + AI 첫 켜기 (25분)

- **참고 리서치:** `_workspace/01_research_ai-basics.md`

### 학습 목표
1. ChatGPT 무료 계정 가입·로그인을 완료한다.
2. 첫 프롬프트를 한국어로 입력해 응답을 받는다.
3. 개인정보 입력 금지 원칙을 청중 모두가 공유한다.

### 입문자 핵심 (필수)

- 채팅창에 "안녕, 나는 공인중개사야"부터 시작
- **"이름·전화번호·정확한 주소 입력 금지" 빨간 슬라이드 1장**으로 모두에게 못박기

### 중급자 심화 (선택)
- ChatGPT vs Claude 같은 질문 비교 데모 (5분)
- 데이터 학습 옵트아웃 설정 경로 (설정 > 데이터 컨트롤 > "Improve the model for everyone" OFF)
- Plus($20) 결제 시점 판단 기준 ("무료 한도가 답답할 때")

### 강의 흐름 (총 25분)
| 시간 | 단계 | 내용 |
|------|------|------|
| 0~5분 | 도입 | 강사 자기소개 + 청중 거수 ("ChatGPT 써본 적 있는 분?") |
| 5~10분 | 개념 | AI는 무엇을 잘하고 못하나 (광고문 ⭕, 정확한 시세 ❌) |

### 실습 시나리오

- **목적:** 어색함 깨기. 매물 정보 없이 가벼운 질문으로 첫 응답 받아보기.
- **예상 출력:** ChatGPT의 친근한 점심 추천 응답.

### 즉시 효용

## 모듈 3: 고객 응대 + 개인정보 마스킹 (35분)

- **참고 리서치:** `_workspace/01_research_customer-service.md`
- **★ 필수 반영 3: 개인정보 마스킹 방법론 (정규식 + 메모장 Ctrl+H)**

### 학습 목표
1. 카톡 문의 원문을 **메모장 Ctrl+H 일괄치환**으로 마스킹할 수 있다.
2. 마스킹된 텍스트를 ChatGPT에 넣어 답변 초안을 받고, 발송 전 사람이 검수하는 워크플로우를 수행한다.
3. **개인정보보호법 제28조의8 국외 이전** 조항과 그 회피 방법(마스킹 = 가명정보 처리)을 이해한다.

### 입문자 핵심 (필수)
- 카톡 원문 메모장에 붙여넣기 → Ctrl+H로 이름·전화번호 치환 → AI에 넣기
- 치환 예시: "김민수" → "고객A", "010-1234-5678" → "010-****-****"
- AI 답변은 **반드시 사람이 검수 후 발송** (공인중개사법 제25조 확인설명 의무)

### 중급자 심화 (선택)
- Python 정규식 패턴 (전화·주민번호·계좌·이메일) 데모
- 클로바노트 통화 녹음 → 자동 요약 → ChatGPT 답례 카톡 워크플로우
- 국내 LLM (HyperCLOVA X, Solar) 선택 시 국외 이전 동의 면제 (이론만, 실습 X)

### 강의 흐름 (총 35분)
| 시간 | 단계 | 내용 |
|------|------|------|
| 0~5분 | 도입 | "고객 실명을 ChatGPT에 넣으면 어떻게 되나" 충격 1슬라이드 |
| 5~12분 | 개념 | 개인정보보호법 제17조·제28조의2·제28조의8 핵심 3조 |
| 12~20분 | 시연 | 카톡 원문 → 메모장 Ctrl+H 마스킹 → ChatGPT 입력 → 답변 검수 |
| 20~30분 | 실습 | 청중 각자: 강사 배포 가상 카톡 원문 1건 마스킹 → 답변 초안 생성 |
| 30~35분 | 정리 | 마스킹 체크리스트 (이름·전화·주소·계좌) 4종 인쇄본 배포 |

### 실습 시나리오
- **배포 원문 (마스킹 대상):**
  ```
  김민수 고객(010-1234-5678)이 역삼동 K오피스텔 1203호 매물 보고 싶다고 함.
  본인 회사 강남파이낸스센터 근무, 출근 도보 가능한지 물음.
  ```
- **메모장 Ctrl+H 치환 작업 (청중 직접):**
  - "김민수" → "고객A"
  - "010-1234-5678" → "010-****-****"
  - "1203호" → "###호"
  - "강남파이낸스센터" → "강남 인근 직장"
- **ChatGPT 입력 후 출력 예시:** 3톤(공손/친근/전문가) 카톡 답변 + 직접 방문 안내 마무리.

### 즉시 효용
**오늘 사무실 돌아가서:** 메모장에 "마스킹 치환 단축키 4종"(이름·전화·호수·직장) 메모 저장. 내일 첫 카톡 문의가 오면 이 워크플로우로 답변 초안을 받아본다.

## 모듈 4: 시장·법령 리서치 + 환각 검증 (30분)

- **참고 리서치:** `_workspace/01_research_market-law.md` (주), `_workspace/01_research_ai-basics.md` (환각 부분)
- **★ 필수 반영 4: 1차 출처 검증 5단계 (AI 환각 회피)**

### 학습 목표
1. **AI 환각 회피 5단계 체크리스트**(법령조문·판례·실거래가·정책시행일·숫자)를 외운다.
2. 호갱노노/디스코로 시세 빠른 조회 후 rt.molit.go.kr에서 1차 검증하는 절차를 수행한다.
3. ChatGPT가 인용한 "공인중개사법 제○조"를 law.go.kr에서 직접 재검색해 실존 확인할 수 있다.

### 입문자 핵심 (필수)
- **5단계 체크리스트:**
  1. 법령 조문 → law.go.kr 직접 검색
