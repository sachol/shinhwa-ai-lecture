## 1. 핵심 도구·서비스 (상위 7개)

### 1.1 국토교통부 실거래가 공개시스템 (rt.molit.go.kr)
- **카테고리:** 1차 공식 데이터(시세)
- **가격:** 무료 (웹 조회), 무료 (공공데이터포털 API 키 신청)
- **한국어:** ★★★★★ (한국어 전용)
- **공인중개사 적합도:** ★★★★★ (시세 검증의 절대 기준점)
- **요약:** 부동산 거래신고법에 따라 신고된 실거래가 자료를 제공. 아파트, 연립/다세대, 단독/다가구, 오피스텔, 토지, 상업·업무용까지 매매·전월세 모두 공개. 2006년 1월 이후 데이터 누적. API는 공공데이터포털(data.go.kr)에서 별도 신청.
- **출처:** https://rt.molit.go.kr/ (2026-05-28 확인), https://www.data.go.kr/data/15126469/openapi.do
- **입문자 핵심:** "ChatGPT가 시세 말하면 무조건 여기서 다시 확인". 동·아파트명·계약월로 검색.
- **중급자 심화:** 공공데이터포털 API 키 + Excel Power Query/Python으로 특정 단지의 월별 시세 추이를 자동 다운로드. 호갱노노보다 1~2주 빠른 1차 데이터.

### 1.2 호갱노노 (hogangnono.com)
- **카테고리:** 2차 가공 시세 플랫폼(아파트 특화)
- **가격:** 무료 (앱/웹), 직방 인수(2018)
- **한국어:** ★★★★★
- **공인중개사 적합도:** ★★★★☆ (아파트 한정, 토지·상가 약함)
- **요약:** 국토부 실거래가를 지도 기반으로 시각화. 거래 일자/가격/층/평형 + 현재 호가 + 전세가 + 과거 시세 추이를 한 화면에 표시. 2025년 "이야기 탭"·실시간 알림 강화.
- **출처:** https://hogangnono.com/ (2026-05-28 확인)
- **입문자 핵심:** 지도에서 단지 클릭 → 시세 흐름 즉시 확인. 고객 상담 시 "이 단지 최근 거래가는…" 한 화면 공유.
- **중급자 심화:** 알림 설정으로 담당 권역 신규 거래 푸시. 단, 호가는 사용자 등록 기반이라 100% 신뢰 금지 — 매매가는 실거래가, 호가는 참고.

### 1.3 디스코 (disco.re)
- **카테고리:** 2차 가공 시세 플랫폼(토지·상가·빌딩 특화)
- **가격:** 무료 (기본), 일부 유료 분석 기능
- **한국어:** ★★★★★
- **공인중개사 적합도:** ★★★★★ (상가/토지 중개 시 필수)
- **요약:** 토지·상가·빌딩 실거래가에 강점. 평당가·차트·필지 단위 분석 제공. 호갱노노가 아파트라면 디스코는 상업용·투자형 부동산.
- **출처:** https://disco.re/ (2026-05-28 확인)
- **입문자 핵심:** 토지·상가 브리핑이면 호갱노노 말고 디스코.
- **중급자 심화:** 인근 필지 거래 이력 + 용도지역 + 건폐율/용적률 시뮬레이션으로 매물 수익성 자료 작성.

### 1.4 국가공간정보포털 / V-World (vworld.kr)
- **카테고리:** 공간정보 1차 API
- **가격:** 무료 (API 키 신청)
- **한국어:** ★★★★★
- **공인중개사 적합도:** ★★★☆☆ (직접 활용은 중급 이상, GIS/지도 임베드 시 강력)
- **요약:** 국토부가 운영하는 공간정보 오픈 API. 2D/3D 지도, 행정구역, 지적, 도로명주소, 건축물 등 레이어를 무료로 호출. 중개사무소 등록현황도 공공데이터포털에서 제공.
- **출처:** https://www.vworld.kr/v4po_main.do (2026-05-28 확인), https://www.data.go.kr/data/15063946/fileData.do
- **입문자 핵심:** 직접 안 써도 됨. 호갱노노/디스코가 내부적으로 이 데이터를 쓴다는 사실만 인지.
- **중급자 심화:** V-World API + Python으로 담당 권역 신규 중개사무소 개업 현황을 월 단위 트래킹(경쟁 모니터링).

### 1.5 법제처 국가법령정보센터 (law.go.kr)
- **카테고리:** 1차 공식 법령·판례
- **가격:** 무료
- **한국어:** ★★★★★
- **공인중개사 적합도:** ★★★★★ (법령 인용의 절대 기준점)
- **요약:** 공인중개사법, 부동산 거래신고 등에 관한 법률, 주택임대차보호법, 상가건물 임대차보호법 등 현행 조문을 무료 제공. 법제처 자체 "Lawbot(지능형 법령검색 시스템)"도 운영하지만 키워드 부분매칭으로 인한 환각 사례 보고됨.
- **출처:** https://www.law.go.kr/ (2026-05-28 확인), https://law.go.kr/LSW/aai/main.do
- **입문자 핵심:** ChatGPT가 조문 번호 인용하면 무조건 이 사이트에서 "제○조의○" 그대로 검색해서 재확인.
- **중급자 심화:** "시행일자별 본문" 기능으로 개정 이전/이후 조문 비교. 정책 변화 영향 분석 시 필수.

### 1.6 korean-law-mcp (GitHub: chrisryugj/korean-law-mcp)
- **카테고리:** Claude/Cursor 연동 MCP 서버
- **가격:** 무료 (오픈소스, 법제처 API 키 필요)
- **한국어:** ★★★★★
- **공인중개사 적합도:** ★★★☆☆ (Claude Desktop 설치 필요, 입문자에겐 진입장벽)
- **요약:** 법제처 42개 API를 17개 MCP 도구로 통합. **LLM 환각 방지 인용검증(verify_citations)** 기능 내장. 판례·조례·법령해석례·행정규칙까지 자연어 질의 가능. 주임법·상임법·부거법 등 부동산 관련 법령 포함. 별도 유사 프로젝트로 SeoNaRu, seo-jinseok, woongaro, rabqatab(LexLink) 등 다수.
- **출처:** https://github.com/chrisryugj/korean-law-mcp (2026-05-28 확인), https://github.com/darjeeling/awesome-mcp-korea
- **입문자 핵심:** 이런 게 있다는 정도만 알면 충분. 설치는 IT에 익숙한 동료에게 부탁.
- **중급자 심화:** Claude Desktop에 설치 후 "공인중개사법 제25조 설명의무 위반 판례 5개 찾아줘"라고 자연어 질의 → 자동으로 법제처 검증된 데이터 회신. impact_map으로 조문 개정 영향 추적, time_travel로 시점별 비교.

### 1.7 NotebookLM (notebooklm.google)
- **카테고리:** 문서 기반 폐쇄형 RAG (Retrieval-Augmented Generation)
- **가격:** 무료 (Google 계정)
- **한국어:** ★★★★☆ (2025 한국어 업데이트 완료)
- **공인중개사 적합도:** ★★★★☆ (법령 PDF·정책 자료 요약에 강함)
- **요약:** 사용자가 업로드한 자료(PDF, 웹페이지, YouTube)만 근거로 답변 생성 → **외부 학습 데이터 기반 환각 위험이 가장 낮음**. 2025년 동영상 개요, 마인드맵, AI 오디오(팟캐스트) 등 추가.
- **출처:** https://notebooklm.google/?hl=ko (2026-05-28 확인), https://support.google.com/notebooklm/?hl=ko
- **입문자 핵심:** "10·15 부동산 대책" PDF를 통째로 업로드 → "주요 변경점 5가지만 표로" 요청. 답변마다 원문 인용 표시.
- **중급자 심화:** 법제처에서 다운로드한 공인중개사법 전문 + 시행령 + 시행규칙 3개 PDF를 한 노트북에 묶고, 매물 상황별 적용 조문을 질의. 일반 ChatGPT보다 인용 정확도 압도적.

## 2. 실제 적용 사례 (4개)

### 사례 2.1: 특정 동 매물 시세 흐름 분석 (마포구 공덕동 A아파트)
- **상황:** 상담 고객이 "공덕동 A아파트 24평 적정 호가가 얼마냐"고 물음. 5분 안에 데이터 자료 작성 필요.
- **사용 도구 조합:** 호갱노노(빠른 시각화) → 국토부 실거래가(검증) → ChatGPT(자료 정리)
- **절차:**
  1. 호갱노노 앱에서 A아파트 검색 → 최근 6개월 84㎡(24평) 실거래가 5건 추출
  2. **rt.molit.go.kr에서 같은 단지·같은 평형 5건을 재조회하여 일치 확인** (호갱노노 데이터 지연 가능성 점검)
  3. ChatGPT에 "다음 5건 거래의 평균/중앙값/최고/최저를 표로 만들고, 거래 시점별 추이 한 줄 코멘트" 요청
  4. 호가는 별도로 "현 시점 호가"로 명시, 실거래가와 구분
- **입력 예시 (3단계 프롬프트):**
  ```
  다음은 마포구 공덕동 A아파트 84㎡ 최근 6개월 실거래(매매)다.
  2025-11: 16.5억(15층) / 2026-01: 16.2억(8층) / 2026-02: 17.0억(20층) / ...
  요청: 1) 표로 정리, 2) 평균·중앙값 계산, 3) 시세 흐름 한 줄, 4) "이는 실거래가이며 호가가 아님" 문구 포함.
  ```
- **출력 예시:** 표 + "최근 6개월 16.2~17.0억 사이 박스권. 2026-02 거래 1건이 상승 신호이나 표본 부족으로 단정 어려움."
- **출처:** 실제 사례 — research-analyst 추정 시나리오 (도구 조합은 검증됨)
- **검증 핵심:** **ChatGPT에 데이터 입력만 시키고 시세 출처는 절대 ChatGPT에서 받지 않음**

### 사례 2.2: 10·15 부동산 대책 영향 — 고객 상담 자료 5분 작성
- **상황:** 2025-10-15 발표된 토지거래허가구역·LTV 강화 대책. 고객이 "내가 사려는 매물에 영향 있냐" 문의.
- **사용 도구 조합:** NotebookLM (정책 원문 요약) + 법제처 (시행일·조문 확인) + 경향신문 Q&A 기사 (실무 해설)
- **절차:**
  1. 국토부 보도자료 PDF 다운로드 → NotebookLM 업로드
  2. "이 대책의 ① 적용 지역 ② 적용 대상 ③ 시행일 ④ 실거주 의무 요건을 표로" 요청
  3. **NotebookLM 답변의 모든 숫자·날짜를 경향신문/한경 1차 보도와 교차확인**
  4. 법제처에서 "주택법 시행령" 등 관련 시행령 개정안 확인 (시행일이 명시되어야 함)
- **출력 예시:** "서울 전 지역 + 경기 12개 지역 토지거래허가구역 지정 / 2년 실거주 의무 / 2025-10-20 계약분부터 적용"
- **출처:** https://www.khan.co.kr/article/202510151656001 (경향신문 10·15 Q&A, 2026-05-28 확인)
- **검증 핵심:** 정책 시행일은 한 글자도 추측 금지. 보도자료·기사·법제처 3중 확인.

### 사례 2.3: 신탁 부동산 임대차 중개 시 설명의무 — 판례 기반 리스크 사전 점검
- **상황:** 신탁등기된 오피스텔 임대차 의뢰가 들어옴. "수탁자(신탁회사) 동의 없이 위탁자(원소유자)와 계약하면 임차인 권리는?" 사전 점검.
- **사용 도구:** korean-law-mcp (또는 대법원 종합법률정보)
- **절차:**
  1. Claude Desktop + korean-law-mcp에 "신탁 부동산 임대차 중개 설명의무 위반 판례"로 질의
  2. 회신된 판례 사건번호를 **대법원 종합법률정보(glaw.scourt.go.kr) 또는 law.go.kr 판례 섹션에서 직접 검색하여 실재 확인**
  3. 실제 사례: 서울북부지법 2024가단111852 (2025-05-13 선고) — 신탁관계 부동산 임대차 중개 시 수탁자 정보 설명의무
- **출력 활용:** 고객에게 "신탁등기 확인 + 수탁자 동의서 필수" 안내, 중개사 본인 분쟁 예방
- **출처:** https://scourt.go.kr/portal/dcboard/DcNewsViewAction.work?cbub_code=000213&gubun=44&seqnum=26957 (2026-05-28 확인)
- **검증 핵심:** **AI가 알려준 판례 사건번호는 100% 대법원 사이트에서 실존 확인**. 미국·한국에서 변호사가 AI 생성 가짜 판례를 법원에 제출했다가 적발된 사례 다수.

### 사례 2.4: 담당 권역 일일 뉴스 모니터링 자동화
- **상황:** 마포·용산 권역 부동산 뉴스를 매일 아침 5분 안에 파악하고 싶음.
- **사용 도구:** 빅카인즈 (bigkinds.or.kr) + ChatGPT 요약
- **절차:**
  1. 빅카인즈에서 "마포구 OR 용산구 + 부동산 OR 재개발 OR 재건축" 키워드, 기간: 어제~오늘 설정 → 기사 목록 다운로드
  2. ChatGPT/NotebookLM에 기사 본문 붙여넣고 "5줄 요약 + 우리 권역에 영향 있는 항목만 ★ 표시" 요청
  3. 빅카인즈 자체 AI 요약 기능도 활용 가능 (2025-03-24 개편)
- **출력 예시:** "1. ★ 마포 공덕역 인근 재개발조합 설립 인가 (○○일보) / 2. 용산 한남뉴타운 분양가 협의 ..."
- **출처:** https://www.bigkinds.or.kr/ (2026-05-28 확인), https://www.kpf.or.kr/front/board/boardContentsView.do?board_id=246&contents_id=df08b9e568964e0ebc978a79574c4801
- **검증 핵심:** 빅카인즈는 1차 언론사 DB라 환각 위험 거의 없음. ChatGPT는 "요약만" 시키고 추가 정보 생성 금지.

## 3. 주의사항·리스크

### 3.1 AI 환각(hallucination) 실제 피해 사례
- **국내 변호사 사건(2024~2025):** 한 지방법원 형사재판부에 변호사가 제출한 **판결 5개가 실제로 존재하지 않음**. 변호사가 'AI를 사용했다'고 인정. 한국형사법무정책연구원 2024 조사: 변호사 4명 중 3명이 법조 AI 사용 경험.
  - 출처: https://www.daeryunlaw.com/trend/10342, https://www.lawtimes.co.kr/opinion/188234
- **미국 변호사 벌금형(2023):** ChatGPT 생성 가짜 판례 제출 변호사 2명 $5,000 벌금.
  - 출처: https://www.lawtimes.co.kr/news/187960
- **경찰 사건:** 경찰이 ChatGPT 사용 중 허위 판례를 인용한 사례 보도.
- **법제처 Lawbot 자체 환각:** 키워드 부분매칭으로 "산안기준규칙" 질의가 국가표준기본법으로 잘못 매칭되는 등 공식 시스템도 100% 신뢰 불가.

### 3.2 공인중개사 관점 환각 회피 5단계 체크리스트
1. **법령 조문:** ChatGPT가 "공인중개사법 제25조"라고 하면 → law.go.kr에서 직접 "공인중개사법 제25조" 검색 → 조문 내용 일치 확인.
2. **판례 사건번호:** ChatGPT가 사건번호 인용하면 → glaw.scourt.go.kr(대법원 종합법률정보) 또는 law.go.kr 판례 검색에서 사건번호 그대로 입력 → 실존 확인.
3. **실거래가:** ChatGPT가 "○○동 ○○아파트 최근 시세는 ○억"이라고 하면 → **무조건 rt.molit.go.kr에서 동일 단지·평형·시기 재조회**. AI는 학습 데이터 시점이 고정.
4. **정책 시행일:** ChatGPT가 "2025년 ○월 ○일부터 시행"이라고 하면 → 국토부 보도자료 + 법제처 시행령 부칙 2중 확인.
5. **숫자 인용 금지:** AI에게 "정확한 숫자(시세, 세율, 면적)"를 묻지 말고, **이미 검증된 숫자를 주고 "정리만" 시킬 것**.

### 3.3 법령·개인정보 관련
- **공인중개사법 제25조 (중개대상물 확인·설명):** 중개사의 설명의무는 AI가 대체 불가. 확인·설명서는 사람이 작성·서명. (조문 번호는 법제처 재확인 필수)
- **공인중개사법 제18조의2 (중개대상물의 표시·광고):** AI 생성 광고문도 동일하게 광고법 적용. 허위·과장 광고 금지.
- **개인정보보호법:** 고객 이름·전화번호·주민번호·임장 기록을 ChatGPT 무료/Plus 일반 채팅에 입력 금지. 학습 데이터로 사용될 수 있음 (Claude·ChatGPT 모두 설정에서 학습 거부 옵션 확인 필요).
- **자본시장법(투자권유 금지):** AI를 이용한 "이 매물 사면 ○% 오른다" 류의 시세 예측·투자 추천은 자본시장법 위반 소지. 시장 분석은 "과거 추이 정리"까지만.

### 3.4 도구별 데이터 시점 차이
- 호갱노노/디스코: 국토부 데이터를 약 1~7일 지연 반영
- ChatGPT/Claude: 학습 시점 이후 데이터는 부재 (웹 검색 기능 활성화해도 부정확 가능)
- **공식 1차 출처(rt.molit.go.kr, law.go.kr)만 "현행" 보장**
