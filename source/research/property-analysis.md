# 매물 분석·광고문 자동 생성 리서치 노트

> 작성: research-analyst | 검토: 미검증 | 최종 수정: 2026-05-28

이 노트는 공인중개사가 AI를 이용해 **매물 광고문·카드뉴스·이미지**를 만들 때 (1) 어떤 도구가 실무에 적합한가 (2) 어떤 표현·이미지가 법령 위반이 되는가 두 축으로 정리했다. 한국 광고법령(공인중개사법, 표시·광고의 공정화에 관한 법률, 2026.1.22. 시행 AI 기본법)을 우선 인용한다.

---

## 1. 핵심 도구·서비스 (상위 6개)

### 1.1 ChatGPT (GPT-4o / GPT-5)
- **카테고리:** 텍스트 생성형 AI (광고문 초안, 매물 설명, 건축물대장 해석)
- **가격:** 무료 (GPT-5 mini 제한) / Plus 월 $20 / Team 월 $25/인
- **한국어:** ★★★★★ (자연스러운 부동산 표현 가능, 단 법령 인용은 환각 위험)
- **공인중개사 적합도:** ★★★★★
- **요약:** 가장 보편적인 텍스트 광고문 생성 도구. 매물 정보(평수, 층, 향, 옵션)를 입력하면 한방·네이버·블로그용 광고문을 톤별로 3~5종 즉시 생성. GPTs(맞춤 챗봇) 기능으로 "우리 사무소 톤"을 저장해 재사용 가능. 한국경제 기사에 따르면 미국 중개업자가 4베드룸 소개글 작성에 1시간 → 5초로 단축한 사례가 있다.
- **출처:** [openai.com/chatgpt/pricing](https://openai.com/chatgpt/pricing) (2026-05 확인), [한국경제 2023-02-03](https://www.hankyung.com/article/202302029753Y)
- **입문자 핵심:** "사진 같은 매물 5개 정보(주소·평수·층·향·가격)를 던지고 '네이버 블로그 광고문 3개 톤별로 써줘'라고 말하기"
- **중급자 심화:** GPTs(맞춤 챗봇)에 사무소 표준 명시사항 체크리스트(소재지, 면적, 가격, 공인중개사 등록번호, 위반건축물 여부)를 시스템 프롬프트로 박아두고 매번 자동 포함

### 1.2 Claude (Sonnet 4.5 / Opus 4)
- **카테고리:** 텍스트 생성형 AI (긴 문서·계약서·건축물대장 PDF 해석에 강점)
- **가격:** 무료 (제한) / Pro 월 $20 / Max 월 $100~200
- **한국어:** ★★★★★ (커뮤니티 평가상 ChatGPT보다 뉘앙스·문장 결이 자연스럽다는 의견 다수)
- **공인중개사 적합도:** ★★★★☆
- **요약:** 200K 토큰 컨텍스트로 건축물대장·등기부등본 PDF를 통째로 넣고 해석 가능. 한국어 글쓰기 결이 ChatGPT보다 부드럽다는 평이 있어 정중한 톤의 임장 안내문·메일에 유리. Artifacts 기능으로 광고문 HTML/카드뉴스 코드 즉시 미리보기.
- **출처:** [namu.wiki Claude](https://namu.wiki/w/Claude), [nxcode.io Claude vs ChatGPT 2026](https://www.nxcode.io/resources/news/claude-vs-chatgpt-2026-which-ai-to-use) (2026-05 확인)
- **입문자 핵심:** "건축물대장 PDF를 그대로 드래그해 넣고 '핵심 3줄 요약, 일반 고객이 알아야 할 주의사항 3가지'만 받기"
- **중급자 심화:** Projects 기능에 우리 사무소 광고 가이드라인 + 표시광고법 조문을 미리 업로드해두면 위반 표현을 사전에 걸러주는 1차 필터로 작동

### 1.3 Gamma
- **카테고리:** AI 프레젠테이션·카드뉴스·웹페이지 생성
- **가격:** Free (400 크레딧, 카드 10장 한도, Gamma 워터마크) / Plus 월 $10 (연 $8/월) / Pro 월 $20 (연 $15/월) / Ultra 월 $100
- **한국어:** ★★★★☆ (60+ 언어 지원, 한국어 폰트는 기본 폰트 한계 — Pro에서 커스텀 폰트 가능)
- **공인중개사 적합도:** ★★★★★
- **요약:** "마포구 신축 오피스텔 매물 카드뉴스 만들어줘"라고 한 줄 입력하면 8~10페이지 카드뉴스를 자동 디자인. 무료 플랜은 한 번에 최대 10장, Plus 25장, Ultra 75장까지. 결과물은 PDF·PPT·PNG·웹페이지 어느 형태로도 내보내기 가능. 카카오톡 채널·블로그·인스타그램 배포에 그대로 사용.
- **출처:** [gamma.app/pricing](https://gamma.app/pricing), [Flowith Gamma Pricing 2026](https://flowith.io/blog/gamma-app-pricing-2026-free-vs-plus-vs-pro/) (2026-05 확인)
- **입문자 핵심:** 무료 플랜으로 충분히 시작 가능. 단 Gamma 로고 워터마크가 박히므로 정식 매물 광고용으로는 Plus 이상 권장
- **중급자 심화:** 텍스트 입력 모드 대신 "Import" 모드로 ChatGPT가 만든 매물 설명서를 붙여넣으면 디자인만 입혀줘서 광고 톤 일관성 확보

### 1.4 Canva (Magic Studio / Magic Design)
- **카테고리:** 디자인·카드뉴스·SNS 광고 (AI 보조)
- **가격:** Free / Pro 월 ₩9,900 (연 ₩99,000) / Business 월 ₩12,900 (연 ₩129,000)
- **한국어:** ★★★★★ (한국어 폰트 가장 다양, 한국 디자이너 템플릿 풍부)
- **공인중개사 적합도:** ★★★★★
- **요약:** 부동산 광고 템플릿 컬렉션 보유. Magic Design은 텍스트 한 줄로 카드뉴스·인스타 릴스 광고 자동 생성. Magic Write로 광고 카피 보조. 한국 부동산 SNS 마케팅에서 가장 널리 사용되는 디자인 도구로, Pro 플랜의 배경 제거(매물 사진의 배경 정리)·매직 리사이즈(인스타·블로그·당근용 비율 일괄 변환)가 실무 가치 큼.
- **출처:** [canva.com/pricing](https://www.canva.com/en/pricing/), [canva.com/web-ads/templates/real-estate](https://www.canva.com/web-ads/templates/real-estate/) (2026-05 확인)
- **입문자 핵심:** 무료 플랜으로 부동산 템플릿만 골라써도 즉시 결과물 나옴. 단 사진 배경제거·매직 리사이즈는 Pro 필수
- **중급자 심화:** Brand Kit에 사무소 로고·색상·폰트 등록 → 모든 광고물 톤앤매너 통일. Magic Write의 톤 옵션(Friendly/Professional/Formal)을 매물 등급별로 구분 사용

### 1.5 ChatGPT 이미지 모드 (GPT-4o image, DALL-E 통합)
- **카테고리:** AI 이미지 생성 (인테리어 시안, 가상 스테이징, 일러스트)
- **가격:** ChatGPT Plus 월 $20에 포함
- **한국어:** ★★★★☆ (프롬프트 한국어 OK, 결과물 내 한국어 텍스트 렌더링은 여전히 약함)
- **공인중개사 적합도:** ★★★☆☆ (광고 표시 의무 주의)
- **요약:** "30평 아파트 미니멀 인테리어 시안" 같은 가상 시안 이미지를 텍스트로 생성. **단 실제 매물 사진처럼 보이게 만들어 광고에 사용하면 표시·광고법 위반.** 가상 스테이징(빈 방에 가구 합성)·카드뉴스 일러스트·블로그 썸네일 용도에 한정 권장.
- **출처:** [openai.com](https://openai.com/chatgpt/pricing) (2026-05 확인)
- **입문자 핵심:** AI 생성 이미지에는 반드시 "AI 생성 이미지(가상 시안)" 워터마크나 캡션 부착. 매물 실사 자리에 두지 말 것
- **중급자 심화:** 2026년 1월 22일 AI 기본법 제31조 시행으로 생성형 AI 사용 광고는 사전 고지 의무 → 광고 하단 고정 멘트 템플릿 구비 필요 (위반 시 과태료 3,000만 원 이하)

### 1.6 Midjourney
- **카테고리:** AI 이미지 생성 (고품질 인테리어·건축 비주얼)
- **가격:** Basic 월 $10 / Standard 월 $30 / Pro 월 $60
- **한국어:** ★★★☆☆ (프롬프트는 영어 권장, 한국어도 작동하나 결과 품질 떨어짐)
- **공인중개사 적합도:** ★★☆☆☆ (보조 도구, 광고 직접 사용 비권장)
- **요약:** 인테리어 분위기 컨셉 비주얼 생성에서는 DALL-E보다 결과물이 미려. 단 한국 부동산 광고에 직접 쓰기엔 (1) 영어 프롬프트 요구 (2) 사진 같은 결과물 → 광고법 위반 위험 (3) 한국 사무소 환경 톤과 안 맞음. 전시·홍보용 배너 이미지 컨셉 잡을 때만 권장.
- **출처:** [bannerbear.com Midjourney vs DALL-E 2025](https://www.bannerbear.com/blog/midjourney-vs-dall-e-a-style-by-style-image-comparison-in-2025/) (2026-05 확인)
- **입문자 핵심:** 처음에는 손대지 말 것. ChatGPT 이미지로 충분
- **중급자 심화:** Discord 기반 UX가 진입 장벽. 인스타 광고 비주얼 차별화가 필요한 사무소만 고려

## 2. 실제 적용 사례 (4개)

### 사례 2.1: 마포구 신축 오피스텔 네이버 블로그 광고문 — ChatGPT
- **상황:** 마포구 합정동 신축 오피스텔 15평형, 풀옵션, 한강뷰. 종전 광고문 작성 1시간 → 5분 단축 목표.
- **사용 도구:** ChatGPT (GPT-4o)
- **입력 예시 (프롬프트):**
  ```
  당신은 서울 마포구 공인중개사 사무소의 광고 카피라이터다.
  아래 매물 정보로 네이버 블로그용 광고문 3개 톤별(전문/친근/감성)로 작성하라.
  
  [매물 정보]
  - 소재지: 서울 마포구 합정동 OOO
  - 종류: 오피스텔
  - 면적: 전용 49.5㎡ (약 15평)
  - 층: 12층 / 25층
  - 향: 남동향, 한강 일부 조망
  - 가격: 보증금 1,000 / 월세 110만 원
  - 옵션: 풀옵션 (냉장고, 세탁기, 에어컨, 가스레인지, 빌트인 가구)
  - 입주: 즉시 가능
  - 공인중개사 등록번호: 11440-2024-OOOOO
  - 위반건축물: 해당 없음
  
  [필수 포함 사항]
  - 소재지, 면적(전용+공용), 가격, 입주가능일, 공인중개사 사무소명+등록번호
  - "한강 전체 조망", "역세권 보장", "확정 수익률" 같은 과장 표현 금지
  - 위반건축물 표기 의무(공인중개사법 제18조의2) 준수
  ```
- **출력 예시 (요약):** 전문/친근/감성 3톤의 광고문 각각 약 250자. 명시사항 자동 포함, 과장 표현 회피. 사용자가 가장 맞는 톤을 선택해 약간 손보면 즉시 게시 가능.
- **출처:** research-analyst 추정 시나리오 (한국경제 [2023-02-03 미국 사례](https://www.hankyung.com/article/202302029753Y) 기반 한국 매물로 재구성)

### 사례 2.2: 광주광역시 구도심 빌라 카드뉴스 8장 — Gamma
- **상황:** 광주 동구 구도심 30년 된 빌라 매매 광고. 인스타·카카오톡 채널 동시 배포할 카드뉴스 필요.
- **사용 도구:** Gamma (Plus 플랜)
- **입력 예시 (프롬프트, Gamma):**
  ```
  공인중개사 카드뉴스 8장 만들어줘. 매물은 광주 동구 OOO 빌라,
  전용 59㎡, 3층/4층, 매매가 1억 8천. 학교·시장 도보 5분.
  톤은 부드럽고 신뢰감 있게. 마지막 장은 사무소 연락처·등록번호.
  ```
- **출력 예시 (구성):**
  1. 표지: "광주 동구 구도심 빌라 매매"
  2. 위치·교통 (도보권 학교·시장)
  3. 면적·구조 (3룸, 전용 59㎡)
  4. 가격·관리비
  5. 내부 사진 (실사진 자리)
  6. 주변 인프라
  7. 매물 명시사항(공인중개사법 제18조의2)
  8. 사무소명·등록번호·전화·카카오톡 채널 QR
- **출처:** research-analyst 추정 시나리오 (Gamma 기능 기반 [gamma.app](https://gamma.app/))

### 사례 2.3: 건축물대장 PDF 해석 → 고객 설명문 — Claude
- **상황:** 부산 해운대 노후 아파트. 건축물대장 PDF에 위반건축물 표기, 무단증축 이력. 고객에게 쉽게 설명 필요.
- **사용 도구:** Claude (Pro)
- **입력 예시 (프롬프트):**
  ```
  첨부한 건축물대장 PDF를 분석해서 일반 고객(부동산 비전공자)이 이해할 수 있게
  3개 섹션으로 정리해줘.
  1) 이 건물의 핵심 정보 (위치, 연면적, 사용승인일, 구조)
  2) 주의해야 할 사항 (위반사항, 미등기, 노후도 관련)
  3) 매수 전 추가로 확인해야 할 항목 체크리스트
  
  법령 해석이 필요한 부분은 "법령 확인 필요"로 표시하고 추측하지 마.
  ```
  *[건축물대장 PDF 첨부]*
- **출력 예시 (요약):** 표 형식의 3섹션 정리. 위반증축 부분은 "건축법 위반사항 — 별도 확인 필요" 표시. 매수자에게 그대로 보내거나 사무소 메모로 활용 가능.
- **출처:** research-analyst 추정 시나리오 (Claude 200K 컨텍스트 [활용](https://www.anthropic.com/news/claude-3-5-sonnet))

### 사례 2.4: 인스타그램 매물 카드 6장 — Canva Magic Design
- **상황:** 인천 송도 신축 아파트 84㎡ 전세 매물. 인스타 정사각형 카드 6장 시리즈 광고.
- **사용 도구:** Canva Pro (Magic Design + Brand Kit)
- **입력 예시 (Magic Design 프롬프트):**
  ```
  Instagram square posts, 6 cards, real estate listing, modern minimal,
  ```

## 4. 입문자/중급자 학습 포인트 요약

### 입문자 핵심 3가지
1. **광고문 생성은 ChatGPT 무료 플랜으로 충분** — 매물 정보 5종(주소·평수·층·향·가격)을 던지고 "네이버 블로그 광고문 3개 톤별로 써줘"라고 시키기만 하면 된다
2. **카드뉴스는 Canva 무료 부동산 템플릿이 시작점** — 디자인 못해도 템플릿만 골라 텍스트만 바꾸면 즉시 결과물
3. **AI가 만든 이미지에는 반드시 "AI 생성" 표시** — 실제 매물 사진처럼 보이게 쓰면 표시광고법 위반 + 2026년 1월부터 AI 기본법 위반(과태료 3,000만 원 이하)

### 중급자 심화 3가지
1. **ChatGPT GPTs / Claude Projects에 우리 사무소 표준 명시사항을 시스템 프롬프트로 고정** — 매번 위반건축물·등록번호·금지 표현 체크 자동화
2. **Gamma의 Import 모드 + ChatGPT 출력 조합** — 텍스트 일관성은 GPT가, 디자인은 Gamma가 책임지는 분업으로 카드뉴스 생산 속도 5배
3. **건축물대장 PDF는 Claude 200K 컨텍스트에 통째로 넣어 3섹션 요약** — 매수자 설명문·임장 메모로 즉시 활용, 단 위반건축물·법령 해석은 "법령 확인 필요" 마커로 환각 방지
