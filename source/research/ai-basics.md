## 1. 핵심 도구·서비스 (6개)

### 1.1 ChatGPT (Free)
- **카테고리:** 텍스트·이미지 생성형 AI (대중성 1위)
- **가격:** 무료 (회원가입 필요, 구글 계정 가능)
- **한국어:** ★★★★★ (한국어 자연스러움 최상위권, 2026년 기준)
- **공인중개사 적합도:** ★★★★☆ (광고문 초안·고객 응대 템플릿에 충분)
- **요약:** 가장 진입장벽이 낮은 AI. 무료 플랜에서도 GPT-5 일부 모델 + 음성 입력 + 이미지 분석이 가능하며, 광고 카피·문자 응답 초안 작성에 즉시 활용 가능. 단, 사용량 한도가 있어 바쁜 시간대에는 일시 제한 발생.
- **출처:** https://chatgpt.com/pricing/ (2026-05-28 확인), https://help.openai.com/ko-kr/articles/6950777-what-is-chatgpt-plus (2026-05-28 확인)
- **입문자 핵심:** 회원가입 → chat.openai.com 접속 → 채팅창에 한국어로 "원룸 광고문 초안 만들어줘"라고 말하는 것부터 시작.
- **중급자 심화:** 무료 한도가 빨리 소진된다면 GPT-5 사용 시점을 광고문·계약 검토 같은 "중요한 한 번"에 집중하고, 일상 질문은 GPT-5 mini로 분리.

### 1.2 ChatGPT Plus (월 $20 / 약 29,000원)
- **카테고리:** ChatGPT 유료 플랜 (개인용 표준)
- **가격:** 월 $20 (한국 결제 시 부가세 포함 약 29,000원)
- **한국어:** ★★★★★
- **공인중개사 적합도:** ★★★★★ (가장 비용 대비 효율이 높은 선택)
- **요약:** GPT-5 사용량 5배 확대, 고급 음성 모드, GPT Store의 커스텀 GPT 무제한 사용, 이미지 생성, 파일/스프레드시트 업로드 분석, 메모리 기능. 공인중개사가 가장 먼저 결제할 한 가지 도구로 권장.
- **출처:** https://chatgpt.com/pricing/ (2026-05-28 확인), https://www.glbgpt.com/hub/kr/chatgpt-plus-subscription-price-in-south-korea-2026-guide/ (2026-05-28 확인)
- **입문자 핵심:** "무료로 써보고 한도가 답답하면 그때 결제"가 정답. 처음부터 결제할 필요 없음.
- **중급자 심화:** Plus는 GPT Store의 커스텀 GPT를 무제한으로 쓸 수 있어, "건축물대장 해석 GPT" "광고문 검수 GPT"를 사전 제작해두면 워크플로우가 단축됨.

### 1.3 ChatGPT Pro (월 $200)
- **카테고리:** ChatGPT 최상위 플랜 (고급 추론·Deep Research·Agent 모드)
- **가격:** 월 $200 (연결제 약 $20 인하 보도 있으나 검증 필요)
- **한국어:** ★★★★★
- **공인중개사 적합도:** ★★☆☆☆ (일반 중개업무에는 과한 사양)
- **요약:** GPT-5 Pro reasoning 무제한, Deep Research(긴 시장 보고서 자동 작성), Agent 모드, 무제한 메시지·이미지 생성. 상업용 부동산 컨설팅·대형 단지 분석 등 "주 1회 깊은 시장 보고서"가 필요한 경우만 고려.
- **출처:** https://chatgpt.com/plans/pro/ (2026-05-28 확인)
- **입문자 핵심:** 처음에는 결제 금지. Plus로 부족함을 명확히 느낀 후에 검토.
- **중급자 심화:** Deep Research 기능으로 "강남구 재건축 동향 30페이지 리포트" 같은 산출물을 1시간 내 받을 수 있어, 투자자 상담 비중이 높은 사무소에서는 ROI가 있음.

### 1.4 Claude (Anthropic) — Free / Pro $20
- **카테고리:** 텍스트 생성형 AI (긴 문서·계약서 처리에 강함)
- **가격:** 무료 / Pro 월 $20 (약 29,000원)
- **한국어:** ★★★★☆ (자연스러움은 ChatGPT와 동급, 일부 표현에서 어색함 잔존)
- **공인중개사 적합도:** ★★★★☆ (계약서·등기부 등본 같은 "긴 문서" 분석에 강점)
- **요약:** ChatGPT 대안. 긴 PDF(계약서, 등기부, 건축물대장 50쪽)를 한 번에 읽고 요약하는 능력이 뛰어남. 답변 톤이 차분하고 과장이 적어 광고법 위반 표현이 적게 생성됨. **Claude Projects**(Pro 이상) 기능으로 "우리 사무소 매물 양식"을 학습시켜 일관된 광고문을 양산할 수 있음.
- **출처:** https://claude.com/pricing (2026-05-28 확인), https://support.claude.com/en/articles/8325606-what-is-the-pro-plan (2026-05-28 확인)
- **입문자 핵심:** claude.ai 가입 → 한국어로 바로 질문. ChatGPT와 거의 동일한 방식.
- **중급자 심화:** Claude Projects에 "공인중개사법 광고 규정", "사무소 매물 광고 양식 5개"를 미리 업로드해두면 매번 프롬프트로 설명할 필요 없음. ChatGPT의 "Custom GPT"보다 설정이 단순.

### 1.5 Gemini (Google) — Free / Pro
- **카테고리:** 구글 통합형 AI (Gmail/Docs/Drive와 연동)
- **가격:** 무료 / Google One AI Premium 월 약 29,000원
- **한국어:** ★★★★☆
- **공인중개사 적합도:** ★★★☆☆ (Gmail/캘린더 사용 비중이 큰 사무소에 유리)
- **요약:** Gmail에 들어오는 매물 문의·임대인 연락을 자동 요약하고, 구글 캘린더에 답사 일정을 자동 등록하는 식의 "구글 생태계 자동화"에 강점. 단독 챗봇 성능은 ChatGPT/Claude와 동급이나, 부동산 도메인 특화 콘텐츠는 ChatGPT가 약간 우세.
- **출처:** https://namu.wiki/w/Gemini(%EC%9D%B8%EA%B3%B5%EC%A7%80%EB%8A%A5%20%EB%AA%A8%EB%8D%B8) (2026-05-28 확인), https://hyperlab.hits.ai/blog/gemini-chatgpt-differences (2026-05-28 확인)
- **입문자 핵심:** Gmail에서 "이 메일에 답장 초안 만들어줘" 한 줄이면 끝. 별도 가입 불필요.
- **중급자 심화:** Google Workspace를 사용하는 사무소라면 Gemini로 "이번 주 답사 일정 정리해서 캘린더에 넣어줘"를 자연어 명령 가능. 무료 한도(2.5 Pro 1일 5회)는 빠르게 소진됨.

### 1.6 ChatGPT 모바일 앱 + 고급 음성 모드 (Advanced Voice)
- **카테고리:** 모바일 음성 인터페이스
- **가격:** ChatGPT Plus 이상 (월 $20)
- **한국어:** ★★★★★ (사투리 일부 인식, 2024년 말 대규모 업데이트로 자연스러움 비약적 향상)
- **공인중개사 적합도:** ★★★★★ (현장에서 손이 안 비는 순간 가장 유용)
- **요약:** 운전 중·답사 중·고객 통화 직후 "방금 본 매물 특징 정리해줘" 같은 음성 입력이 가능. 50여 개 언어 인식, 한국어 자연 대화 가능. 외국인 고객과의 통역 보조로도 사용 가능 (단, 법적 효력 있는 통역은 별도 인가 필요).
- **출처:** https://chatgpt.com/features/voice/ (2026-05-28 확인), https://www.tech42.co.kr/%EC%98%81%ED%99%94-her-%EA%B0%99%EC%9D%80-%EA%B3%A0%EA%B8%89-%EC%9D%8C%EC%84%B1%EB%AA%A8%EB%93%9C-%EC%B1%97gpt-%EC%82%AC%EC%9A%A9%ED%95%98%EB%8A%94-%EB%B0%A9/ (2026-05-28 확인)
- **입문자 핵심:** App Store/Play Store에서 "ChatGPT" 검색 → 공식 앱 설치 → 마이크 아이콘. 그게 전부.
- **중급자 심화:** 답사 종료 직후 음성으로 "방금 본 마포구 합정동 25평 신축, 보증금 5천/월 90, 남향, 분리형, 2분 거리 합정역. 카톡 보낼 매물 소개문 만들어줘"라고 말하면 30초 안에 문구가 나옴.

## 2. 실제 적용 사례 (4개)

### 사례 2.1: 처음 ChatGPT 깔고 5분 안에 매물 광고문 1개 뽑기
- **상황:** 50대 공인중개사 K씨. ChatGPT 첫 사용. 마포구 망원동 신축 빌라 광고를 평소 30분 걸려 손으로 작성.
- **사용 도구:** ChatGPT (무료 플랜)
- **입력 예시 (프롬프트):**
  ```
  너는 한국 공인중개사의 광고 카피 도우미야.
  공인중개사법 광고 규정을 지켜서, 과장 없이 사실만으로 작성해줘.

  매물 정보:
  - 위치: 서울 마포구 망원동 (망원역 도보 7분)
  - 면적: 전용 18평
  - 구조: 방2 화1 거실
  - 가격: 전세 4억 2천
  - 특징: 2024년 준공, 남향, 엘리베이터, 주차 1대 가능

  요청:
  1. 네이버 부동산 등록용 100자 소개문
  2. 카카오톡 발송용 5줄 요약
  3. 과장된 표현(초역세권, 풀옵션, 최저가 등) 사용 금지
  ```
- **출력 예시:** "망원역 도보 7분 거리의 2024년 준공 신축 빌라입니다. 전용 18평, 방2/화1 구조에 남향으로 채광이 좋습니다. 엘리베이터와 1대 주차가 가능합니다." + 카톡용 5줄 (생략)
- **출처:** research-analyst 추정 시나리오 (스킬 기반 작성, 실제 사례 미확보)

### 사례 2.2: Claude Projects에 우리 사무소 양식 미리 학습
- **상황:** 강남구 중개법인. 매물 광고문 표준 양식이 있고, 직원 3명이 매번 양식 맞추는 데 시간 소비.
- **사용 도구:** Claude Pro + Projects
- **입력 예시:** Claude Project에 ① 사무소 매물 광고 양식 5개 PDF, ② 공인중개사법 제18조의2 광고 규정 텍스트, ③ "절대 사용 금지 표현 리스트" 1장을 업로드. 이후 "이 매물 정보로 광고문 만들어줘"만 입력.
- **출력 예시:** 매번 동일한 톤·구조·법령 준수 표현으로 광고문 자동 생성. 신입 직원도 시니어와 동일한 품질.
- **출처:** research-analyst 추정 시나리오 (Claude Projects 공식 기능 기반, 실제 사례 미확보) — https://support.claude.com/en/articles/8325606-what-is-the-pro-plan (2026-05-28 확인)

### 사례 2.3: 운전 중 답사 메모를 음성으로 정리
- **상황:** 오전 10시 답사 3건 연속. 메모할 시간 없음. 운전하며 다음 답사지로 이동.
- **사용 도구:** ChatGPT 모바일 + 고급 음성 모드
- **입력 예시 (음성):** "방금 본 매물 정리할게. 합정동 25평 신축, 보증금 5천에 월 90, 남향, 풀옵션은 아니고 빌트인 냉장고와 세탁기만 있어. 단점은 엘리베이터 없음. 다음 약속은 11시 망원동."
- **출력 예시:** 텍스트로 자동 정리 → "[합정동] 25평 신축 / 보5천·월90 / 남향 / 빌트인 냉장고·세탁기 / 엘리베이터 없음 / 다음: 11시 망원동" 사무실 도착 후 그대로 매물장에 복사.
- **출처:** research-analyst 추정 시나리오 (고급 음성 모드 공식 기능 기반)

### 사례 2.4: 외국인 임차 문의 카톡 답장 초안
- **상황:** 영어로 들어온 임차 문의. 사무소 직원 영어 자신 없음.
- **사용 도구:** ChatGPT (Free)
- **입력 예시 (프롬프트):**
  ```
  외국인 고객이 보낸 영어 문의를 한국어로 번역하고,
  답장 초안을 영어로 작성해줘.

  고객 문의: "Hi, I'm looking for a studio near Gangnam station, under 1.5M won monthly rent. Can you help?"

  답장에 포함할 내용:
  - 가능 매물 확인 가능
  - 보증금 수준(deposit) 안내 필요
  - 거주 시작일 문의
  - 한국에서의 외국인 임대차 시 신분증/비자 필요 안내
  ```
- **출력 예시:** 정중한 영어 답장 초안 (생략). 직원이 검토 후 발송.
- **출처:** research-analyst 추정 시나리오 (실무 빈도 높은 유형)

## 3. 주의사항·리스크

### 3.1 개인정보 입력 금지 (최우선)
- **금지 항목:** 고객 실명·주민번호·정확한 주소·전화번호·계좌번호·신분증 사본 이미지.
- **이유:** ChatGPT/Claude 등 외산 AI는 한국 클라우드 외 서버에 입력값이 저장될 수 있고, 일부 플랜은 학습 데이터로 활용될 수 있음. 이는 개인정보보호법 위반 소지 (특히 제28조의8 국외 이전 동의 의무, 2024.3.15 시행).
- **대안:** 고객 식별 정보는 "고객A", "임차인1"로 마스킹하여 입력.
- **출처:** https://www.boannews.com/media/view.asp?idx=119138 (2026-05-28 확인), https://www.pipc.go.kr/np/cop/bbs/selectBoardArticle.do?bbsId=BS074&mCode=C020010000&nttId=10362 개인정보보호위원회 (2026-05-28 확인)
- **검증 필요:** OpenAI의 "데이터를 학습에 사용하지 않음" 설정(설정 > 데이터 컨트롤 > "Improve the model for everyone" OFF)을 켰는지 확인 권장.

### 3.2 환각(Hallucination) — 법령·실거래가는 반드시 원본 확인
- **위험 패턴:** ChatGPT가 "공인중개사법 제○○조에 따르면..."이라며 **실재하지 않는 조문**을 자신 있게 인용하는 경우 발생.
- **대표 사례:** 2023년 미국 변호사가 ChatGPT가 만든 가짜 판례 6건을 법원에 제출해 제재받음.
- **공인중개사 적용:** "이 광고가 광고법 위반인지" "이 거래에 양도세가 얼마인지" 같은 질문은 ChatGPT 답변을 1차 참고로만 쓰고, 반드시 법제처 국가법령정보센터(law.go.kr), 국세청 홈택스, 한국부동산원 실거래가 공개 시스템에서 교차 확인.
- **출처:** https://namu.wiki/w/%EC%9D%B8%EA%B3%B5%EC%A7%80%EB%8A%A5%20%ED%99%98%EA%B0%81 (2026-05-28 확인)

### 3.3 광고법 위반 표현이 AI 출력에 섞일 위험
- AI는 매력적인 카피를 만들도록 학습되어 "초역세권", "최저가", "완벽한", "유일한" 같은 과장 표현을 자연스럽게 생성함.
- 공인중개사법 제18조의2 및 같은 법 시행규칙은 "부당한 표시·광고"를 금지함. **AI 출력을 그대로 게시하지 말고, 반드시 사람이 검수**.
- **검증 필요:** 공인중개사법 제18조의2 최신 개정 조문 (2024~2025 개정 여부 확인 필요)

### 3.4 음성 입력 시 주변 대화 녹취 우려
- 고급 음성 모드 사용 중에는 마이크가 켜져 있어, 의도치 않게 고객·동료의 대화가 녹음·전송될 수 있음.
- 고객 미팅 중에는 음성 모드 사용 자제. 답사 후 차량 1인 환경에서만 사용 권장.

## 부록: 공인중개사가 "처음 깔고 어떻게 말 걸어야 하나"

가장 짧은 3단계:

1. **자기소개 1문장** — "나는 서울에서 일하는 공인중개사야."
2. **상황 1문장** — "마포구 망원동 25평 빌라 광고문을 만들어야 해."
3. **요청 1문장** — "네이버 부동산용 100자 소개문 초안 만들어줘. 과장 표현은 쓰지 마."

이게 끝. 영어·전문 용어 필요 없음. 한국말로 사장님·부장님께 말하듯이.

## 부록: "무엇을 묻지 말아야 하나"

1. **고객 실명·주민번호·정확한 주소** — 절대 입력 금지.
2. **"이 거래 양도세 정확히 얼마야?"** — 환각 위험. 세무사 또는 홈택스에서 확인.
3. **"이 광고가 법 위반이야?"** — 판단의 최종 책임은 사람. AI는 1차 점검만.
4. **"오늘 강남구 실거래가 알려줘"** — 무료 ChatGPT는 실시간 데이터 없음. 국토부 실거래가 공개시스템 직접 확인.
