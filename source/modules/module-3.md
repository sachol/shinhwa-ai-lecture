## 학습 목표 (다시 한 번)

- **입문자:**
  1. 메모장 Ctrl+H 일괄치환만으로 고객 이름·전화·호수를 마스킹할 수 있다
  2. 마스킹된 텍스트를 ChatGPT에 넣어 카톡 답변 초안 3개 톤을 받는다
  3. 발송 전 사람이 직접 검수하는 습관(공인중개사법 제25조)을 갖는다

- **중급자:**

  3. 클로바노트·다글로·에이닷 노트의 통화 후 자동 요약을 워크플로우에 끼워넣는다

### Slide 3-6: 입문자용 마스킹 방법 — 메모장 Ctrl+H

### Slide 3-9: 중급자용 — 정규식 패턴 4종 (참고)

**본문 (슬라이드에 표시):**
```
휴대전화: 01[016789][-\s]?\d{3,4}[-\s]?\d{4}
주민번호: (\d{6})[-\s]?\d{7}
이메일:   [\w\.-]+@[\w\.-]+\.\w+
계좌:     \d{3,6}[-\s]\d{2,4}[-\s]\d{4,7}
```
- 입문자: **이 슬라이드는 메모만**
